# Pedehour.github.io

Ce dépôt contient le site statique pour Pedehour.

Pour publier le site à l'URL utilisateur racine (https://fjykyuy.github.io/), renomme le dépôt en `Fjykyuy.github.io` :

1. Va dans le dépôt sur GitHub.
2. Clique sur **Settings** (Paramètres).
3. Dans **Repository name**, remplace `Pedehour.github.io` par `Fjykyuy.github.io` puis confirme.

Après le renommage, GitHub Pages publiera automatiquement le contenu (index.html) à l'adresse https://fjykyuy.github.io/. Attends quelques minutes si nécessaire.

Si tu préfères garder le nom actuel, l'URL publique sera une page de projet : `https://fjykyuy.github.io/Pedehour.github.io/` (ou vérifier Settings → Pages pour l'URL exacte).
